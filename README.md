
# Меблерія

## Проєкт
**Меблерія** - це сучасний інтернет-магазин меблів, створений для тих, хто цінує комфорт, стиль та якість.  
Наш сайт допомагає користувачам знайти ідеальні рішення для дому, поєднуючи елегантний дизайн із функціональністю.

---

## Завдання проєкту
Мета проєкту - представити колекцію стильних і зручних меблів, які перетворять будь-який простір.  
Сайт демонструє каталог продукції, забезпечує зручну навігацію та привабливу візуальну подачу.


## Tech Stack

**Client:** HTML, CSS, JS

**Server:** Node.js, Vite
